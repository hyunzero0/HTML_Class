태그의 속성

공통속성 : html의 모든 태그에 작성하는 것 id, class, style 개별속성 : 특정태그에서 사용하는 속성 type, src

공통속성

id : 페이지 내에서 유일한 값을 설정하는 것, 태그를 지칭할 때 사용하는 속성 *중복 금지(강제는 아님) class : 페이지 내에서 태그들을 한 개 그룹으로 묶을 때 지정하는 속성 *중복 허용 style : 태그에 css속성을 설정할 수 있게 하는 속성

나는 p id가 test야

나는 h1 id가 test야

클래스가 test1인 h4

클래스가 test1인 p

안녕 나를 클릭해봐

속성을 설정할 때는 속성값에 대해 반드시 ""이나 ''으로 표시해준다.

스타일을 style속성으로 적용할 수도 있다.
